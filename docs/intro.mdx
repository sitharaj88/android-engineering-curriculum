# Android Development — Complete Guide

This program is the **most comprehensive Android curriculum** you can follow as
a self-learner, student, or working professional. It covers every layer of a
production app — from the Kotlin language itself to publishing on Google Play —
using the patterns, libraries, and tools that real engineering teams use today.

<Callout kind="tip" title="What makes this guide different">
We don't stop at "how" — every concept is explained with the **why** behind it.
You'll understand *why* `viewModelScope` is preferred over `GlobalScope`, *why*
`StateFlow` replaced `LiveData`, *why* Compose recomposes the way it does, and
*why* baseline profiles cut startup time by 40%. The reasoning is what makes
you senior.
</Callout>

## Why learn Android in 2026?

Android runs on **3+ billion active devices** worldwide and remains the
dominant mobile OS in every market except the US. With the rise of **Kotlin
Multiplatform**, **Wear OS**, **Android TV**, **foldables**, and **AI-powered
on-device experiences via ML Kit**, Android engineering is more relevant — and
more rewarding — than ever.

Modern Android development is also **far more pleasant** than it used to be.
Java is gone. XML layouts are giving way to declarative Compose. Coroutines
make async code readable. Hilt removes DI boilerplate. The platform finally
feels modern — and this guide teaches you that modern stack from the ground
up.

## Prerequisites

You don't need any Android experience. You **do** need:

- **Basic programming knowledge** in any language (variables, loops, functions, conditionals)
- A computer with **8 GB+ RAM** running Windows, macOS, or Linux
- **Curiosity** and willingness to build real projects

We install Android Studio together in Module 02. Bring your own machine and a
willingness to break things — that's how you learn.

## How to use this guide

<Callout kind="info" title="Recommended pace">
- **Self-learner**: 8–12 hours per week → finish in ~6 months.
- **Student**: 4–6 hours per week → finish in ~9 months alongside coursework.
