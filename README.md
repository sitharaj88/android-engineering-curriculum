# Android Development Training

A comprehensive, production-focused Android development curriculum covering everything from Kotlin fundamentals to publishing on Google Play. Built with Docusaurus and deep-explanation MDX.

- **13 modules · 73+ topics · 34 weeks · 120h content**
- Kotlin · Jetpack Compose · MVVM · Clean Architecture · Hilt
- Room · Retrofit · Coroutines · Flow · Firebase
- Testing · Performance · Play Store publishing · KMP

## Local development

```bash
npm install
npm start            # http://localhost:3000/android-development-training/
npm run build
```

## Deploy

GitHub Pages via the included workflow. Set `url` and `baseUrl` in `docusaurus.config.ts` for your repo.
